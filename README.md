# Calculadora Científica en C++

## Descripción del Programa

Este programa implementa una **calculadora científica básica** que se ejecuta en consola y permite realizar operaciones matemáticas fundamentales y avanzadas mediante un menú interactivo.

Incluye validación de entradas y manejo de errores para evitar operaciones inválidas, como división entre cero, raíces de números negativos o entradas no numéricas.

---

## Funcionalidades

- Menú interactivo en consola.
- Operaciones básicas:
  - Suma
  - Resta
  - Multiplicación
  - División
- Operaciones avanzadas:
  - Factorial
  - Potencia
  - Raíz cuadrada
- Validación de entradas:
  - No permite valores no numéricos.
  - Control de errores matemáticos.
- Manejo de errores mediante excepciones.
- Ejecución continua hasta que el usuario decide salir.

## Instrucciones de Uso

Clonar o descargar el repositorio:
```bash
git clone https://github.com/mariomendez22/Programacion_dinamica_calculadora_cientifica.git
```
Compilar el programa:
```bash
g++ calculadora_cientifica.cpp -o calculadora
```

Ejecutar el programa:
```bash
./calculadora
```
